# mac_changer
Graphical MAC changer tool using Python and Qt5 Designer
## Features

- Graphical user interface (GUI) for easy interaction.
- Customizable design using Qt5 Designer.
- Easy to use.

## Prerequisites

- Python 3.6+
- PyQt5
